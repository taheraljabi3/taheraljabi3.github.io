خطوات ربط جهاز بصمة من نوع Hikvision

تعرف على الخطوات اللازمة لربط جهاز بصمة من نوع Hikvision بقاعدة البيانات وبرنامج المزامنة...

خطوات ربط جهاز بصمة من نوع Hikvision:

أولاً: تحميل وتثبيت البرامج اللازمة:
  • تحميل برنامج SQL Server Express 2014.
  • تحميل وتثبيت برنامج SQL Server Management Studio.
  • تحميل برنامج IVMS-4200 من الموقع الرسمي.
  • تحميل برنامج Desktop_SmartHR.

ثانياً: تثبيت برنامج SQL Server Management Studio:
  • قم بتثبيت البرنامج بطريقة بسيطة وسهلة كما هو موضح في الصور.

ثالثاً: فتح SQL Server Management Studio:
  • افتح البرنامج، واضغط على New Query.
  • قم بإضافة قاعدة البيانات باستخدام الاسكريبت واضغط على Execute.
  • تأكد من المسار الصحيح لإنشاء قاعدة البيانات.

رابعاً: فتح Sql Server 2014 Configuration Manager:
  • قم بتمكين الإعدادات المطلوبة، ثم أعد تشغيل خدمة SQL Server.

خامساً: تثبيت برنامج IVMS-4200:
  • افتح البرنامج واختر Time & Attendance ثم اختر Third-Party Database.
  • أدخل البيانات اللازمة (اسم قاعدة البيانات، اسم المستخدم، كلمة المرور) واضغط على حفظ.

سادساً: تثبيت برنامج Desktop_SmartHR:
  • قم بإدخال رابط الـ API واسم المستخدم وكلمة المرور.
  • قم بتهيئة قاعدة البيانات كما هو موضح في الصورة.

سابعًا: رفع بيانات البصمة:
  • لرفع البيانات من جهاز البصمة، انتقل إلى صفحة Attendance Record وقم بعمل استيراد للبيانات.

خطوات ربط جهاز البصمة ببرنامج ZKTeco

اكتشف الخطوات الأساسية لربط جهاز البصمة ببرنامج ZKTeco Attendance Management. يشمل ذلك تحميل وتثبيت البرامج اللازمة وضبط الإعدادات...

خطوات ربط جهاز البصمة ببرنامج ZKTeco:

أولاً: تحميل وتثبيت البرامج اللازمة:
  • تحميل برنامج SQL Server Management Studio (SSMS 2019).
  • تحميل برنامج SQL Server Express 2019.
  • تحميل برنامج Desktop_SmartHR.
  • تحميل برنامج ZKTeco Attendance Management.

ثانياً: تثبيت البرامج وضبط الإعدادات في SQL Server:
  • بعد تثبيت SQL Server Management Studio، نفتح البرنامج ونضغط على "New Query".
  • ننشئ قاعدة بيانات جديدة من خلال الضغط على "Databases" واختيار "New".

ثالثاً: ضبط قاعدة البيانات وربطها ببرنامج ZKTeco:
  • نفتح برنامج ZKTeco Attendance Management ونذهب إلى قسم "Database Option".
  • نضيف اسم قاعدة البيانات التي أنشأناها سابقًا ونتأكد من الاتصال الصحيح.

رابعاً: ضبط الإعدادات في برنامج Desktop_SmartHR:
  • نقوم بإدخال رابط API واسم المستخدم وكلمة المرور لربط النظام السحابي بالبرنامج.

خامساً: إضافة الموظفين وتسجيل البصمة:
  • نفتح برنامج ZKTeco Attendance Management ونضيف الموظفين بشكل يدوي.
  • نقوم بتسجيل بصمات الدخول والخروج ونختار النوع المناسب (دخول أو خروج).

روابط مفيدة:
  • رابط لشرح جداول الحضور.
